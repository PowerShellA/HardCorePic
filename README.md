# HardCorePic
超硬核的图片防文字识别的工具
在家闲来无事写的小程序。C++编写的基于OPENCV的图片处理工具，防文字识别，原理是对图片进行三角函数畸变。在此基础上还能进行其他变换。代码参数可调。

# 使用方法
将要进行处理的图片文件拖拽到exe文件上，程序会在源图片文件目录生成输出文件。
